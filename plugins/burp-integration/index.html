Burp Integration

Get input from Burp

Burp -> Extender -> Add Jaeles-burp.py as python extensions.

Get your JWT from this API and fill it into Jaeles Tab in Burp then you’re good to go. Click save button if you want to store JWT persistence.

Default log will be store in ~/.jaeles-burp/config.json
