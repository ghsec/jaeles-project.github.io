Understand Signatures

Take a look at this repo for more examples.

Each signature basically split in two part and written in YAML format.

  • Info
  • Request builder

There’re 3 types of signature: single, list, fuzz

Info

Some information about the signature to index in the DB

# ID to index a signature
id: common-01-01
# Type of signature default is "single"
type: list
# Some info about signature
info:
  name: PHP Debug bar
  # those feilds below are optional
  # default is "Info"
  risk: High
  tech: general
  os: general

Request Builder

Everything in a signature have template pattern {{.variable_key}} will be replace to generate request.

There’re some default template variable: {{.BaseURL}}, {{.URL}}, {{.Host}}, {{.Path}}, {{.Extension}}, {{.oob}}

# variables only used when signature type is "list"
variables:
  - ssrf: [google.com,example.com]

# Each element in requests key will be generate and sending separately
requests:
  - method: GET
    redirect: true
    path: >-
     {{.BaseURL}}/confluence/plugins/servlet/gadgets/makeRequest?url=https://127.0.0.1:443@{{.ssrf}}
    headers:
      - User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:55.0) Gecko/20100101 Firefox/55.0
      - X-Atlassian-Token: no-check
    detections:
      - >-
        StatusCode() == 200 && (StringSearch("response", "google.com") || StringSearch("response", "example.com"))

More advanced request builder

Use raw request as a template

requests:
  - method: GET
    redirect: true
    detections:
      - >-
        StatusCode() == 200 && (StringSearch("response", "google.com") || StringSearch("response", "example.com"))
    # Request raw (make sure you're using "|" in this raw key)
    raw: |
      POST /search.php?test=query HTTP/1.1
      Host: {{.Host}}
      Content-Length: 25
      Cache-Control: max-age=0
      Origin: http://{{.Host}}
      Upgrade-Insecure-Requests: 1
      Content-Type: application/x-www-form-urlencoded
      User-Agent: Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_6) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/79.0.3927.0 Safari/537.36
      Accept: text/html,application/xhtml+xml,application/xml;q=0.9,image/webp,image/apng,*/*;q=0.8,application/signed-exchange;v=b3
      Referer: http://{{.Host}}/
      Accept-Encoding: gzip, deflate
      Accept-Language: en-US,en;q=0.9
      Connection: close

With Origin request to use Origin Detection API.


id: sensitive-01-01
type: list

# Original request to compare
origin:
  method: GET
  redirect: true
  path: >-
    {{.BaseURL}}

variables:
  # we can put list of it in a file and load it
  - sensitive: /tmp/sensitive_path.txt

requests:
  - method: GET
    redirect: true
    path: >-
     {{.BaseURL}}/{{.stats}}

    detections:
      - >-
        StatusCode() == 200 && StringSearch("response", "Secret")
      - >-
        StatusCode() == 200 && !StringSearch("response", "Not Found") && Math.abs(ContentLength() - OriginContentLength() > 1000)
